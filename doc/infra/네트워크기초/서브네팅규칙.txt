서브넷을 나누기 위한 작업

1.Host 또는 서브넷 수를 파악한 후 SubNetmask를 변경한다
2.변경된 1 bit부분을 2진수로 순서대로 쓴다
3.나온수에 Host ID에 모두 0bit를 입력하면 서브넷의 address가 나온다
4.Host ID부분을 모두 1bit처리 하면 subnet Broadcast 주소이다.  


네트워크 수에 따라 나누는 공식
------------------------------

2^n >= 나누고 싶은 Network의 갯수
 
n은 Host 필드에서 가져온 Bit의 갯수

또다른 공식

호스트 수에 따라 나누는 공식
------------------------------

2^n -2 >= 나누고 싶은 호스트의 갯수
 
여기서 -2를 해주는 이유는 브로드캐스트와 NeworkID로 사용할 IP 두개를 빼주는 것이다
여기서 구하고자하는 Host의 갯수는 컴퓨터에 부여할 수 있는 Host ID 갯수이다.
따라서 컴퓨터에 부여 불가능한 브로드캐스트와 Network ID의 IP 두개를 빼주는 것이다.
