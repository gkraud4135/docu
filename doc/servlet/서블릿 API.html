서블릿 API

1> 서블릿은 두 패키지

javax.servlet과 javax.servlet.http의 클래스와 인터페이스를 사용한다

javax.servlet패키지는 프로토콜 독립적인 서블릿을 지원하기 위한 클래스를 포함한다

이들 클래스는 javax.servlet.http 패키지의 클래스에 확장(상속)되어 HTTP고유의 기능이 부가된다

최상의 레벨 패키지 이름은 javax이며, 이는 서블릿 API가 표준에 대한 확장임을 나타낸다

 

2> 모든 서블릿은 javax.servlet.Servlet인터페이스를 구현해야 한다

대부분의 서블릿은 두 클래스 javax.servlet.GenericServlet 혹은 javax.servlet.http.HttpServlet중 의 하나를 확장 함으로써 서블릿을 구현한다.

HttpServlet자체는 HTTP고유의 기능이 부가된 GenericServlet의 서브(파생)클래스이어야한다.

 

3> 정규 자바와 달리 , 애플릿처럼 서블릿도 main()메서드를 갖지 않는다

대신, 서블릿의 특정 메서드는 요청을 처리하는 프로세서 안에서 서버에 의해호출된다

서버가 서블릿에 요청을 전달할 때마다 , 서블릿의 service()메서드가 호출된다

 

4> GenericServlet은 요청을 처리하기 위해 자신의 service()메서들를 오버라이드 해야한다

service()메서드는 두 파라미터들(요청 객체와 응답 객체)을 받아들인다.

요청객체는 서블릿 요청에 대한 정보를제공하고

응답객체는 응답을 반환한기 위해 사용된다

 

5>  HttpServlet 은 일반적으로 service()메서드를 오버라이드 하지 않는다

대신 , Get 요청을 다루기 위해 doGet()을 오버라이드하며

Post 요처을 다루기 위해 doPost()을 오버라이드한다

HttpServlet은 이들 모두 혹은 어느 한쪽 메서드에 대해 다룰 필요가 있는 요청 유형에 따라 오버라이드 할 수 있다

 

6> javax.servlet 안의 ServletRequest 와 ServletResponse 클래스는 Generic서버 요청과 응답에 대한 접근을 제공하는  반면

javax.servlet.http 안의 HttpServletRequest 와 HttpServletResponse 는 HTTP요청과 응답에 대한 접근을 제공한다

Post방식은 방대한 데이터를 전송 할 때( 데이터는 http 내부에 변수값을 숨겨서 보낸다)

import java.io.*;

import javax.servlet.*;

import javax.servlet.http.*;

 

public class ServletTest01 extends HttpServlet{

     

      // doGet() 오버라이딩

      // HttpServletRequest req : 서버로 요청시 (요청 객체)

      // HttpServletResponse res: 클라이언트로 (응답 객체)

      // doGet() : 디폴트 이다

      // doGet() : 소량 데이터 전송시 사용

     

      public void doGet(HttpServletRequest req, HttpServletResponse res)

            throws IOException,ServletException

      {

            //html 페이지의 표준 MIME컨텐츠 타입 지정

            res.setContentType("text/html;charset=euc-kr");

           

            PrintWriter out = res.getWriter();// out 응답 객체 생성

           

            //클라이언트로 응답

            out.println("<html>");

            out.println("<body>");

           

            out.println("<big>HelloWorld</big><br>");

            out.println("<big>안녕하세요</big><br>");

 

            out.println("</body>");

            out.println("</html>");

            out.close();

 

      }//doGet

}//class
