MIME(인터넷 표준) - Multipurpose Internet Mail Extention

=> 인터넷 전자 우편으로 텍스트 이외의 ,음성, 동영상 등

다양한 테이터를 보내기 위한 표준이다

따라서 jsp컨테이너가 클라이언트에게 문서를 보낼때

이 문서는 어떤 형식의 문서이다 라고 지정해 주어야한다

 

res.setContentType("text/html;charset=euc-kr");//html 페이지의 표준 MIME컨텐츠 타입 지정

 

"text/html" => text 문서를 html 로 보내겠다 

"text/plain" => text문서 보통 문서로 보내겠다

"text/*"

"*/*"

"image/jpg"

"image/gif"

"text/xml"

web.xml 에 설정이 되어 있다

 

======한글처리 ====================================

 

res.setContentType("text/html;charset=euc-kr");//charset=euc-kr 웹으로 응답 할때 한글 처리 하고

 

 

server.xml 에서 9000 포트 검색하여

URIEncoding="EUC-KR"  : 클라이언트로 넘어온 한글을 처리하기 위하여 설정 ( get 방식 )

req.setCharacterEncoding("euc-kr") : 클라이언트로 넘어온 한글을 처리하기 위하여 설정 (post 방식 )

getBytes() : String 를 default character 로 인코딩화 하여 flatform 에 따라 byte로 convert 하여 byte배열에 저장

 

 

 

 

 

Get 방식은 소량 데이터를 전송 할때 사용 한다

    요청 URL에 문자들이 열로서 덧붙여서 전송된다

    중요 id , pwd 는 Get 방식으로 보내면 안된다

 

Post 은 방대한 데이터를 전송할 때 사용한다

     서버에 데이터를 전송할때 데이터를 숨겨서 보낸다

     중요한 데이터를 post 방식으로 한다
